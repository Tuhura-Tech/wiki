You can start writing SQL right now, without installing anything! Let's explore how to use an online SQL editor to practice your database skills.

## What is an Online SQL Editor?

An online SQL editor is a website where you can write and run SQL code right in your browser! It's perfect for:

- Learning SQL without setup
- Testing quick queries
- Experimenting and having fun
- Practicing anywhere - even on a tablet!

## Creating Your Own Tables

You can create your own tables too! Here's a fun example with pets:

```sql
-- Create a pets table
CREATE TABLE Pets (
  pet_id INTEGER PRIMARY KEY,
  name TEXT,
  type TEXT,
  age INTEGER,
  color TEXT
);

-- Add some pets
INSERT INTO Pets (pet_id, name, type, age, color)
VALUES 
  (1, 'Fluffy', 'Cat', 3, 'Orange'),
  (2, 'Buddy', 'Dog', 5, 'Brown'),
  (3, 'Whiskers', 'Cat', 2, 'Gray'),
  (4, 'Max', 'Dog', 7, 'Black'),
  (5, 'Goldie', 'Fish', 1, 'Gold');

-- See all pets
SELECT * FROM Pets;
```

**Output:**
| pet_id | name     | type | age | color  |
|--------|----------|------|-----|--------|
| 1      | Fluffy   | Cat  | 3   | Orange |
| 2      | Buddy    | Dog  | 5   | Brown  |
| 3      | Whiskers | Cat  | 2   | Gray   |
| 4      | Max      | Dog  | 7   | Black  |
| 5      | Goldie   | Fish | 1   | Gold   |

**Find all cats:**
```sql
SELECT * FROM Pets WHERE type = 'Cat';
```

**Output:**
| pet_id | name     | type | age | color  |
|--------|----------|------|-----|--------|
| 1      | Fluffy   | Cat  | 3   | Orange |
| 3      | Whiskers | Cat  | 2   | Gray   |

**Find pets older than 3:**
```sql
SELECT name, type, age FROM Pets
WHERE age > 3
ORDER BY age;
```

**Output:**
| name  | type | age |
|-------|------|-----|
| Buddy | Dog  | 5   |
| Max   | Dog  | 7   |

## Sample Tables for Challenges

### Video Games Table

```sql
CREATE TABLE Games (
  game_id INTEGER PRIMARY KEY,
  title TEXT,
  platform TEXT,
  rating INTEGER,
  release_year INTEGER
);

INSERT INTO Games VALUES
  (1, 'Minecraft', 'PC', 10, 2011),
  (2, 'Mario Kart', 'Switch', 9, 2017),
  (3, 'Pokemon', 'Switch', 9, 2019),
  (4, 'Roblox', 'PC', 8, 2006),
  (5, 'Among Us', 'Mobile', 8, 2018);

SELECT * FROM Games;
```

**Output:**
| game_id | title     | platform | rating | release_year |
|---------|-----------|----------|--------|--------------|
| 1       | Minecraft | PC       | 10     | 2011         |
| 2       | Mario Kart| Switch   | 9      | 2017         |
| 3       | Pokemon   | Switch   | 9      | 2019         |
| 4       | Roblox    | PC       | 8      | 2006         |
| 5       | Among Us  | Mobile   | 8      | 2018         |

**Find games with rating 9 or higher:**
```sql
SELECT title, rating FROM Games
WHERE rating >= 9;
```

**Output:**
| title      | rating |
|------------|--------|

| Pokemon    | 9      |

### Superheroes Table

```sql
CREATE TABLE Superheroes (
  hero_id INTEGER PRIMARY KEY,
  name TEXT,
  superpower TEXT,
```

**Output:**
| hero_id | name        | superpower        | strength_level | team           |
|---------|-------------|-------------------|----------------|----------------|
| 1       | Spider-Man  | Web-slinging      | 85             | Avengers       |

```sql
FROM Superheroes
```

| Wonder Woman | Super Strength    | 95             |

## Pro Tips for Online SQL

### Tip 1: Use Comments
```sql
-- This is a comment - SQL ignores it!
SELECT * FROM Pets; -- Get all pets
```

### Tip 2: Format Your Code
Make it readable with proper spacing:
```sql
-- Good formatting
SELECT name, age
FROM Customers
WHERE country = 'USA'
ORDER BY age;
```

### Tip 3: Save Your Work
Copy your favorite queries into a text file so you don't lose them!

### Tip 4: Experiment!
Don't be afraid to try things! You can't break anything - just refresh the page to start over!

### Tip 5: One Query at a Time
The online editor runs all your code at once. If you want to run just one query, delete or comment out the others!
