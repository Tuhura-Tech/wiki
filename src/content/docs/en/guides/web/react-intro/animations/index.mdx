Animations make apps feel alive. Start small with CSS transitions and toggle styles from React.

## Color transition example

```jsx
import { useState } from 'react'

function ColorBox() {
  const [on, setOn] = useState(false)
  return (
    <div>
      <div style={{
        width: 120,
        height: 80,
        background: on ? '#16a34a' : '#334155',
        transition: 'background-color 300ms ease',
        borderRadius: 8,
        marginBottom: 8
      }} />
      <button onClick={() => setOn(o => !o)}>{on ? 'Turn off' : 'Turn on'}</button>
    </div>
  )
}
```
