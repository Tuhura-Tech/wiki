This is a guide on how to create a 'Clicker' style game in [Godot](https://godotengine.org/)

If you are unfamiliar with Godot, check out the [Godot basics](/game-design/godot/basics) doc


:::note[Version]
This guide is up-to-date with Godot 4.6 stable official release but will most likely work with any 4.x release.
:::

## What *IS* a Clicker Game?

A Clicker game, otherwise known as an incremental game, is a type of game in which the player can gather points by, well, clicking! They can then spend these points on upgrades that increase the power of clicks, or generate points without clicking.

While this might sound simple, or even boring, there's a reason this type of game has taken off! Simply, seeing a number go up REALLY high is fun!

For an example of one of the first (and most famous) clicker style games, check out [Cookie Clicker](https://orteil.dashnet.org/cookieclicker/)

## What we'll be making

We'll be making our very own clicker game, however, its theme is up to you! You can make your clicker about whatever you want. You could be clicking to get points, chairs, octagons, or anything else!

My game will be a Godot themed clicker, so feel free to do that too if you can't think of anything just yet.

At its core the gameplay loop of our game will be:

1. Click the object to get points
2. Spend those points to improve how quickly we get points

Following in Cookie Clickers footsteps, there'll be two kinds of upgrades we can buy:
1. Upgrades - These can only be bought once and provide a bonus percentage, either overall or to a specific Helper.
2. Helpers - These generate points automatically, without us having to click.

Because the game will be relatively simple, we'll also be spending some time adding something game developers call 'Juice'


## What's Juice?

Juice is a broad term for things in a game that don't change how it plays, but change how it *feels.* Things like screen shake, particles, bouncy UI, all contribute to a game feeling more fun without changing how it works.

So instead of just making a game that works, we'll be trying to make a game that works *and* feels extra fun to play! If you're confused by the idea of Juice, don't worry, it'll make more sense once we add some.
