## What are Logic Gates?

Logic gates are the building blocks of digital circuits.

They take binary inputs (`0` or `1`) and produce a binary output based on a rule.  
Think of them as **decision-makers** for computers and electronic systems.

---

## Why Logic Gates Matter

Logic gates are used in:

- Computers and processors
- Calculators
- Traffic lights
- Robots and sensors
- Games and control systems

Every “decision” a computer makes is built from combinations of logic gates.

Output is `1` only if **all inputs are `1`**

## XOR (Exclusive OR)

**Rule:**  
Output is `1` **only when the inputs are different**  
XOR outputs `1` when **exactly one input is `1`**

## Gates with Inverted Output (Bubble)

In logic diagrams, a small circle (○) at the output means **NOT (inversion)**.

Adding a circle does not create a new rule —  
it simply **reverses the output** of the original gate.

## Combined Logic Circuit – Example

<img
  src="/src/assets/electronics/eg.png"
  alt="Combined logic circuit example"
  style={{ maxWidth: "100%", margin: "1.5rem 0" }}
/>

The circuit shown combines multiple basic logic gates into a single decision structure.

Inputs are processed in **parallel branches** and merged in later stages to produce a final output.

For the input state:

- A = 1, B = 1  
- C = 0, D = 1  
- F = 0, G = 1  
- H = 1, I = 1  

the circuit evaluates as follows:

| Signal | Gate | Input Values | Output |
|-------|------|--------------|--------|
| P | AND | 1, 1 | 1 |
| Q | OR | 0, 1 | 1 |
| R | XOR | 0, 1 | 1 |
| S | NAND | 1, 1 | 0 |
| T | OR | 1, 1 | 1 |
| U | AND | 1, 0 | 0 |

An XOR gate produces the final output: X = T ⊕ U = 1 ⊕ 0 = 1
