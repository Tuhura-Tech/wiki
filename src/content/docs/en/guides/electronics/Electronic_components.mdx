## Overview

This page lists commonly used electronic components found in physical-computing projects.

Components are grouped by function and are frequently used with microcontrollers such as RP2040-based boards and Adafruit development platforms.

##### Resistor Colour Coding (Color Bands)

Resistor values are commonly indicated using colored bands printed on the resistor body.  
Each color represents a numeric value, a multiplier, or a tolerance. Both 4-band and 5-band resistor formats are widely used.
